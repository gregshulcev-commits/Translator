# `app.py`

## Назначение

Точка входа приложения.

## Что делает

- читает CLI-аргумент с путём к PDF;
- создаёт `AppConfig`;
- создаёт runtime-каталоги;
- гарантирует наличие встроенного SQLite-словаря;
- настраивает логирование;
- загружает плагины;
- создаёт сервисы и главное окно.

## Важная деталь

`app.py` не скачивает внешние словари автоматически. За это отвечает отдельный инструмент `tools/install_default_dictionaries.py`.
