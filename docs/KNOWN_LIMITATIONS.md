# Известные ограничения

## Документы

- поддерживаются только PDF с текстовым слоем;
- OCR отсутствует;
- текст внутри изображений не переводится;
- canvas не даёт системного выделения текста как в нативном PDF viewer.

## Словари

- встроенный словарь — технический и ограниченный;
- общий словарь FreeDict устанавливается отдельным шагом во время установки;
- нет GUI для управления словарными паками.

## UI

- интерфейс остаётся прагматичным и лёгким, а не полноценным modern desktop shell;
- тема улучшена, но Tkinter всё ещё ограничивает глубину визуального оформления.

## Платформы

- MVP протестирован как Linux desktop приложение;
- Android-версия не входит в эту итерацию.

## Перевод

- только словарный EN -> RU;
- без контекстного нейронного перевода предложений;
- лучший перевод зависит от качества и порядка записей в словарном паке.
