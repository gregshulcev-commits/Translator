# `main_window.py`

## Назначение

Главное окно Tkinter-приложения.

## Основные части

- toolbar;
- canvas для страницы PDF;
- scrollbars;
- нижняя карточка краткой словарной подсказки;
- statusbar.

## Поведение

- открывает PDF через `DocumentService`;
- обрабатывает клик мышью;
- прокручивает страницу колесом мыши;
- подсвечивает выбранное слово;
- обновляет нижнюю карточку перевода;
- работает с поиском и навигацией по результатам.

## Изменение в v2

Панель перевода перенесена вниз и упрощена до краткой справки:

- лучший перевод;
- пример использования;
- альтернативные переводы.
