# Пользовательское руководство

## Поддерживаемые документы

Приложение умеет открывать:

- PDF с текстовым слоем;
- TXT;
- FB2.

## Открытие документа

1. Нажмите **Открыть** на панели инструментов.
2. Либо используйте меню **Файл → Открыть документ…**.
3. Выберите `*.pdf`, `*.txt` или `*.fb2`.
4. После загрузки документ появится в общем scrolling viewer-е.

## Перевод слова

1. Прокрутите документ к нужному месту.
2. Нажмите на незнакомое слово.
3. Слово выделится оранжевой рамкой.
4. Внизу появится краткая справка:
   - заголовок/форма слова;
   - лучший перевод;
   - пример употребления;
   - альтернативные переводы.

Если перевода нет, приложение покажет проверенные candidate forms.

## Прокрутка документа

- колесо мыши — прокрутка всего документа вверх/вниз;
- кнопки `◀` и `▶` — переход к предыдущей / следующей странице;
- `PageUp` / `PageDown` — переход между страницами.

## Масштабирование

- кнопки `-` и `+` на toolbar;
- `Ctrl + колесо мыши` — быстрый zoom под курсором.

Текущий лимит масштаба: **50% — 600%**.

## Поиск

1. Введите слово или фразу в строку `Поиск`.
2. Нажмите `Найти`.
3. Используйте `Пред.` и `След.` для переключения между совпадениями.
4. Найденный фрагмент подсвечивается зелёной рамкой.

## Управление словарями

### Через верхнее меню

Меню **Словари** позволяет:

- скачать общий FreeDict EN→RU;
- подключить готовый SQLite-словарь;
- импортировать CSV-глоссарий;
- импортировать FreeDict TEI.

### Через контекстное меню

Правый клик по viewer-у или нижней панели подсказки открывает те же действия со словарями.

## Изменение размера интерфейса

Используйте меню **Вид**:

- `Интерфейс крупнее`
- `Интерфейс мельче`
- `Сбросить размер интерфейса`

Настройка сохраняется автоматически.

## Где лежат словари

Встроенный словарь:

```text
data/starter_dictionary.sqlite
```

Пользовательские словарные паки:

```text
~/.local/share/pdf_word_translator_mvp/dictionaries/
```

## Где лежат настройки и логи

Настройки:

```text
~/.local/share/pdf_word_translator_mvp/settings.json
```

Логи:

```text
~/.cache/pdf_word_translator_mvp/logs/
```
