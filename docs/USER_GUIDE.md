# Пользовательское руководство

## Поддерживаемые документы

Приложение умеет открывать:

- PDF с текстовым слоем;
- TXT;
- FB2.

## Открытие документа

1. Нажмите **Открыть** на панели инструментов.
2. Либо используйте меню **Файл → Открыть документ…**.
3. Выберите `*.pdf`, `*.txt` или `*.fb2`.
4. После загрузки документ появится в общем scrolling viewer-е.

## Перевод слова

1. Прокрутите документ к нужному месту.
2. Нажмите на незнакомое слово.
3. Слово выделится оранжевой рамкой.
4. Внизу появится компактная справка:
   - форма слова / транскрипция;
   - лучший перевод;
   - контекстный перевод или короткий пример;
   - альтернативные переводы.

Если перевод не найден, приложение покажет проверенные candidate forms.

## Направление перевода

Переключить направление можно:

- кнопкой `EN → RU` / `RU → EN` на toolbar;
- через меню **Перевод → Направление**.

Смена направления сразу влияет на следующие клики по словам. Если слово уже выделено, приложение переоценит перевод для текущего выбора.

## Прокрутка документа

- колесо мыши — прокрутка всего документа вверх / вниз;
- кнопки `◀` и `▶` — переход к предыдущей / следующей странице;
- `PageUp` / `PageDown` — переход между страницами.

После исправлений v5/v6 viewer больше не возвращает вас к старому highlight при обычной прокрутке: можно выбрать слово на 3-й странице и спокойно читать дальше на 5-й и ниже.

## Масштабирование

- кнопки `-` и `+` на toolbar;
- `Ctrl + колесо мыши` — быстрый zoom под курсором.

Текущий лимит масштаба: **50% — 800%**.

## Контекстный перевод

Во второй строке нижней панели приложение может показывать контекстный перевод предложения, в котором встретилось слово.

Режим выбирается в меню **Перевод → Контекстный перевод**:

- `Отключено` — показывается только словарная подсказка;
- `Argos (офлайн)` — локальный нейронный контекстный перевод;
- `LibreTranslate` — HTTP API;
- `Yandex Cloud` — HTTP API.

Настройки выбранного провайдера открываются через **Перевод → Настроить текущий провайдер…**.

### LibreTranslate

Для `LibreTranslate` укажите URL сервера и, при необходимости, API key.

### Yandex Cloud

Для `Yandex Cloud` заполните:

- `Folder ID`;
- `API key` **или** `IAM token`.

### Argos (офлайн)

Для `Argos` нужны две отдельные вещи:

1. установленный Python runtime `argostranslate`;
2. установленная модель нужного направления.

Если чего-то не хватает, приложение теперь не падает в неясную ошибку, а показывает понятную подсказку в нижней панели и в окне настроек провайдера.

## Офлайн-модели Argos из GUI

### Где открыть

Меню:

- **Перевод → Офлайн-модели Argos…**

Окно показывает:

- направление (`EN → RU`, `RU → EN`);
- статус модели;
- версию найденного пакета;
- имя пакета Argos;
- пояснение, что именно ещё нужно сделать.

### Сценарий 1. Скачать модель из сети

1. Сначала установите optional dependency:

```bash
source .venv/bin/activate
python -m pip install -r requirements-optional.txt
```

2. Откройте **Перевод → Офлайн-модели Argos…**.
3. Нажмите **Обновить список из сети**.
4. Выберите нужное направление.
5. Нажмите **Установить выбранную модель**.
6. Нажмите **Выбрать Argos** или переключите провайдер вручную.

### Сценарий 2. Импортировать локальный `.argosmodel`

Если файл уже скачан заранее:

1. откройте **Перевод → Офлайн-модели Argos…**;
2. нажмите **Импортировать .argosmodel…**;
3. выберите локальный файл;
4. переключитесь на `Argos (офлайн)`.

Это удобно для полуофлайн-сценария: модель скачивается один раз отдельно, затем приложение работает локально без сети.

### Сценарий 3. Быстрая справка

В этом же меню остаётся пункт **Как установить Argos…**, который показывает:

- команду установки optional dependency;
- путь через GUI;
- CLI-команды `--list`, `--from-lang/--to-lang`, `--file`.

## Поиск

1. Введите слово или фразу в строку `Поиск`.
2. Нажмите `Найти`.
3. Используйте `Пред.` и `След.` для переключения между совпадениями.
4. Найденный фрагмент подсвечивается зелёной рамкой.

## Управление словарями

### Каталог словарей

Откройте **Словари → Каталог словарей…**.

Каталог показывает:

- название пакета;
- направление;
- категорию;
- источник;
- описание.

Из окна каталога можно:

- установить выбранный пакет;
- открыть страницу источника в браузере;
- обновить список.

### Импорт словарей

Меню **Словари** и правый клик позволяют:

- подключить готовый SQLite-пак;
- импортировать CSV-глоссарий;
- импортировать FreeDict TEI;
- установить FreeDict для текущего направления.

## Изменение размера интерфейса

Используйте кнопки `A−` и `A+` на toolbar или пункты меню **Вид**:

- увеличить размер интерфейса;
- уменьшить размер интерфейса;
- сбросить размер к значению по умолчанию.

Настройка сохраняется автоматически.

В v5/v6 также исправлено масштабирование строк в списках: названия словарей и Argos-моделей теперь не вылезают за пределы строки при крупном UI font size.

## CLI-помощники

### Проверить и поставить Argos модели

```bash
source .venv/bin/activate
PYTHONPATH=src python tools/install_argos_model.py --list
PYTHONPATH=src python tools/install_argos_model.py --from-lang en --to-lang ru
PYTHONPATH=src python tools/install_argos_model.py --file /path/to/model.argosmodel
```

### Импорт словаря

```bash
PYTHONPATH=src python tools/import_dictionary.py my_glossary.csv ~/.local/share/pdf_word_translator_mvp/dictionaries/my_glossary.sqlite --format csv
```
