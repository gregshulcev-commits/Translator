# `token_splitter.py`

## Назначение

Разбиение compound token'ов на отдельно кликабельные части.

## Что умеет

Если текстовый токен содержит разделитель между буквенно-цифровыми частями, модуль делит его на несколько `WordToken` и пропорционально распределяет исходный прямоугольник.

Примеры:

- `diagnostic/measurement`
- `diagnostic\\measurement`

## Зачем это нужно

Без этого клик по левой части compound token'а выделял бы весь токен целиком. В v4 приложение может отдельно выбрать `diagnostic` и `measurement`.
