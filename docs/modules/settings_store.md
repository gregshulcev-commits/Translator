# `settings_store.py`

## Назначение

Сохранение и загрузка пользовательских UI-настроек в JSON.

## Что хранится

- размер шрифта интерфейса;
- направление перевода;
- выбранный контекстный провайдер;
- URL / API key для LibreTranslate;
- credentials для Yandex Cloud.

## Что важно в v8

- значения по-прежнему нормализуются перед сохранением;
- `settings.json` теперь сохраняется атомарно через временный файл и `os.replace()`;
- на POSIX файл получает права `0600`.

Это не превращает JSON-файл в полноценное secret storage, но заметно уменьшает риск случайного раскрытия настроек провайдера другим пользователям системы.
