# `tools/install_default_dictionaries.py`

## Назначение

Установочный сценарий словарей по умолчанию.

## Что делает

1. создаёт runtime-каталоги;
2. гарантирует наличие встроенного технического словаря;
3. пытается скачать общий FreeDict EN-RU словарь;
4. конвертирует его в SQLite и кладёт в runtime-каталог словарей.

## Поведение при ошибке сети

Если скачать общий словарь не удалось, инструмент:

- печатает предупреждение;
- оставляет рабочим встроенный технический словарь;
- не делает приложение неработоспособным.
