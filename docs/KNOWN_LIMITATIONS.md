# Ограничения MVP

## 1. Только PDF с текстовым слоем

Сканированные PDF и текст на картинках пока не поддерживаются.

## 2. Стартовый словарь ограничен

Словарь в архиве — не полный общеязыковой EN->RU словарь, а стартовый пакет для MVP.

## 3. Нет системного выделения текста поверх страницы

Страница рендерится как изображение, поэтому системное выделение текста мышью не реализовано.

## 4. Нет OCR

Для OCR уже подготовлена архитектурная точка расширения, но реализация отложена.

## 5. Нет Android-сборки

Архив содержит только desktop MVP.

## 6. Нет нейронного перевода фраз

Перевод в MVP словарный.

## 7. Нет истории слов

История легко добавляется отдельным модулем, но сейчас отсутствует.
