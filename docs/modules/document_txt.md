# `document_txt.py`

## Назначение

Document plugin для plain-text файлов.

## Как работает

- читает UTF-8 текст;
- разбивает его на paragraphs по пустым строкам;
- передаёт paragraphs в `TextDocumentSession`.

## Формат

- расширение: `.txt`
- кодировка: UTF-8

## Ограничения

- сложная разметка plain-text файлов не поддерживается;
- нет автоопределения кодировки.
