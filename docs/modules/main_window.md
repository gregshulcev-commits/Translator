# `main_window.py`

## Назначение

Главное desktop-окно проекта на Tkinter.

В `v9` модуль отвечает не только за viewer и клик по слову, но и за связку с новым окном настроек и GUI-сценариями установки словарей/Argos.

## Основные части окна

- menu bar;
- toolbar;
- scrolling canvas с lazy rendering страниц;
- нижняя карточка словарной подсказки;
- строка статуса;
- контекстное меню по словарям;
- диалоги каталога словарей;
- GUI manager для Argos;
- точка входа в отдельное окно `SettingsDialog`.

## Что важно в `v9`

### 1. Настройки вынесены в отдельное окно

`MainWindow` больше не держит все provider/UI настройки в одном временном popup.

Теперь он:

- создаёт `SettingsDialog`;
- переиспользует одно и то же окно настроек;
- умеет открывать его сразу на нужной вкладке (`general`, `libretranslate`, `yandex`, `argos`, `dictionaries`).

### 2. Toolbar стал проще

С toolbar убраны отдельные кнопки управления размером UI-шрифта. Размер интерфейса теперь меняется в окне настроек, а на панели остались:

- открытие документа;
- навигация по страницам;
- zoom документа;
- переключение направления;
- быстрый доступ к настройкам;
- быстрый доступ к словарям;
- поиск.

### 3. Argos поддерживает сценарий «без терминала»

Из главного окна доступны:

- установка optional runtime для Argos;
- обновление списка моделей;
- установка модели из сети;
- импорт локального `.argosmodel`;
- выбор Argos как активного провайдера.

### 4. Словари управляются из GUI

`MainWindow` остаётся точкой входа для:

- каталога словарей;
- импорта SQLite;
- импорта CSV;
- импорта FreeDict TEI;
- обновления `DictionaryService` после изменения набора паков.

В `v9` эти действия также доступны через вкладку «Словари» в окне настроек.

## Основные сценарии

- открытие PDF / TXT / FB2 через `DocumentService`;
- непрерывный скролл документа;
- zoom через кнопки и `Ctrl + колесо мыши`;
- клик по слову и запуск click-to-translate workflow;
- подсветка выбранного слова;
- поиск по документу;
- запуск контекстного перевода через `ContextTranslationService`;
- открытие окна настроек и сохранение параметров без ручного редактирования JSON.

## Ключевые методы

- `render_document()` — перестраивает layout страниц;
- `_refresh_visible_pages()` — рендерит только видимые и соседние страницы;
- `process_click_at_page_coords()` — запускает перевод по клику;
- `_start_context_translation()` — передаёт короткий скрытый контекст провайдеру;
- `show_dictionary_catalog()` — каталог устанавливаемых словарей;
- `show_argos_model_manager()` — окно управления офлайн-моделями;
- `install_argos_runtime_from_gui()` — GUI-установка optional Argos runtime;
- `show_settings_window()` — открывает/фокусирует `SettingsDialog`.
