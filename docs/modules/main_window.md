# ui/main_window.py

## Назначение

Главное окно и пользовательское взаимодействие.

## Отвечает за

- toolbar;
- canvas со страницей;
- правую панель перевода;
- поиск;
- навигацию по страницам;
- визуальное выделение слова и поискового результата.

## Важно

В модуле нет логики словаря и PDF-парсинга — он работает только через services.
