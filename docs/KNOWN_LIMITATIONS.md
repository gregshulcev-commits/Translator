# Известные ограничения

## Документы

- PDF должен содержать текстовый слой;
- OCR отсутствует;
- текст внутри изображений не переводится;
- TXT / FB2 рендерятся как synthetic pages, поэтому системное выделение текста там не реализовано.

## Viewer

- lazy rendering сильно уменьшает фризы при zoom, но один очень тяжёлый page render всё ещё может ощущаться на больших страницах и высоких масштабах;
- viewer остаётся лёгким и прагматичным, а не полноценным native PDF shell;
- thumbnails и overview-панель страниц пока отсутствуют.

## Словари

- качество перевода зависит от состава установленных паков и их порядка;
- FreeDict зависит от доступности сети на этапе установки словаря;
- нет удаления словарей из GUI, только добавление / подключение.

## UI

- интерфейс построен на Tkinter, поэтому внешний вид улучшен стилями, но не дотягивает до полноценного modern native shell;
- настройка интерфейса сейчас ограничена размером шрифта;
- установка Argos-модели из GUI пока без отдельного progress bar.

## Платформы

- MVP протестирован как Linux desktop приложение;
- текущий Tkinter-based GUI не переносится на Android без отдельной UI-ветки.

## Перевод

- словарный перевод слова работает офлайн;
- контекстный перевод зависит от выбранного optional провайдера;
- Yandex / LibreTranslate требуют внешнюю настройку и сеть;
- Argos требует отдельной установки optional dependency и модели;
- Argos manager сейчас ориентирован на направления **EN ↔ RU**;
- удаление Argos-моделей из GUI пока не реализовано.
