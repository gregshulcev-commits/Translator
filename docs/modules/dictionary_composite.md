# `dictionary_composite.py`

## Назначение

Объединяет несколько `DictionaryPlugin` в один логический словарь.

## Поведение

- плагины опрашиваются по порядку;
- первый найденный результат возвращается в UI;
- количество статей суммируется для отображения статистики.

## Использование в MVP

Composite plugin объединяет:

- встроенный технический глоссарий;
- пользовательские SQLite-паки.
