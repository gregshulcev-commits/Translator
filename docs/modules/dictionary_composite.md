# `dictionary_composite.py`

## Назначение

Логическая обёртка над несколькими словарными пакетами.

## Алгоритм lookup

1. фильтрует child-паки по направлению;
2. обходит их по приоритету загрузки;
3. возвращает первое найденное совпадение;
4. если совпадения нет, возвращает fallback `LookupResult`.

## Что изменилось в v4

Composite plugin теперь:

- реализует полный `DictionaryPlugin` API;
- умеет проверять `supports(direction)`;
- принимает `direction` в `lookup()`.
