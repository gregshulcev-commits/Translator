# `text_normalizer.py`

## Назначение

Лёгкая нормализация английских словоформ без больших NLP-зависимостей.

## Что делает

- чистит пунктуацию по краям;
- приводит к нижнему регистру;
- генерирует кандидатные формы:
  - plural -> singular
  - `-ed`
  - `-ing`
  - possessive forms
  - дефисные формы

## Почему это важно

Даже при словарном переводе пользователь кликает по форме `configured`, `systems`, `drivers`, а lookup должен находить `configure`, `system`, `driver`.
