# `dictionary_service.py`

## Назначение

Тонкий facade над активным `DictionaryPlugin`.

## Что делает

- выполняет `lookup()`;
- отдаёт количество словарных статей;
- отдаёт количество подключённых словарных паков;
- умеет hot-swap'ить активный plugin после установки нового словаря.

## Особенность v3

При `replace_plugin()` сервис пытается корректно закрыть предыдущий словарный backend, если он поддерживает `close()`.
