# Руководство пользователя

## Открытие документа

1. Нажмите **«Открыть PDF»**.
2. Выберите PDF-файл с текстовым слоем.
3. Первая страница откроется автоматически.

## Перевод слова

1. Кликните мышью по слову в документе.
2. Слово будет выделено прямоугольником.
3. Справа появятся:
   - исходное слово;
   - словарная форма;
   - транскрипция;
   - лучший перевод;
   - альтернативные переводы;
   - контекст из документа.

## Навигация по страницам

- кнопка `◀` — предыдущая страница;
- кнопка `▶` — следующая страница;
- клавиша `PageUp` — предыдущая;
- клавиша `PageDown` — следующая.

## Масштаб

- `-` — уменьшить;
- `+` — увеличить.

## Поиск

1. Введите запрос в поле **«Поиск»**.
2. Нажмите **«Найти»**.
3. Используйте **«Пред.»** и **«След.»** для перехода по совпадениям.

## Горячие клавиши

- `Ctrl+O` — открыть PDF;
- `Ctrl+F` — перейти к строке поиска.

## Где лежит лог

Приложение пишет лог в XDG cache directory пользователя:

```text
~/.cache/pdf_word_translator_mvp/logs/app.log
```

## Что делать, если слово не найдено

В MVP это нормальный сценарий.

Справа будет показано:

- что слово не найдено в стартовом словаре;
- какие словоформы были проверены.

Следующий этап развития — подключение более полного словаря.
