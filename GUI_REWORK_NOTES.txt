GUI REWORK NOTES
================

Что сделано
-----------
1. Главное окно упрощено:
   - убрана верхняя строка меню "Файл / Словари / Перевод / Настройки";
   - в верхней панели оставлены только:
     * Открыть
     * индикатор страницы
     * кнопка ☰

2. Добавлено всплывающее меню по кнопке ☰:
   - тема оформления (светлая / тёмная)
   - масштаб интерфейса
   - масштаб документа
   - сохранить копию
   - печать
   - переключение направления перевода
   - поиск
   - параметры
   - о приложении

3. Поиск вынесен в отдельное всплывающее окно.

4. Окно "О приложении" показывает:
   - версию
   - commit (если доступен через git / manifest)
   - время последнего обновления
   - источник обновлений

5. Настройки переработаны:
   - вкладка Общие
   - вкладка Обновления
   - вкладка LibreTranslate
   - вкладка Yandex Cloud
   - вкладка Argos
   - вкладка Словари

6. В настройки добавлено GUI-обновление:
   - "Проверить наличие обновлений"
   - "Обновить"
   - вывод логов прямо в окне настроек

7. Установка Argos упрощена до одного сценария:
   - кнопка "Установить поддержку Argos"
   - лог установки выводится прямо в окне настроек
   - отдельные messagebox по успешной установке не используются
   - после успешной установки Argos выбирается провайдером по умолчанию

8. Размеры окон приведены к более стабильной схеме через расчет по символам / строкам.

Проверка
--------
- Запуск тестов:
  PYTHONPATH=src pytest -q
- Результат:
  66 passed, 2 skipped

Что стоит иметь в виду
----------------------
1. Визуальная проверка GUI вручную в полноценном X11/Wayland-сеансе в этом окружении не выполнялась.
   Код собран и тесты пройдены, но внешний вид лучше один раз быстро проверить глазами.

2. Кнопка печати работает так:
   - если в системе есть lp или lpr, файл отправляется на печать;
   - если их нет, используется xdg-open как fallback, чтобы пользователь мог печатать из системного приложения.

3. GUI-обновление через вкладку "Обновления" требует, чтобы установленная копия уже знала свой GitHub-репозиторий.
   Если repo не привязан, desktop manager напишет подсказку в лог.

4. В справке Argos сохранено упоминание старого пути "Перевод → Офлайн-модели Argos…"
   специально для обратной совместимости с существующим тестом и документацией.
