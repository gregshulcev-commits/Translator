# Известные ограничения

## Документы

- PDF должен содержать текстовый слой;
- OCR отсутствует;
- текст внутри изображений не переводится;
- TXT/FB2 рендерятся как synthetic pages, поэтому системное выделение текста там не реализовано.

## Viewer

- непрерывный scroll реализован рендером всех страниц документа в canvas, без page virtualization;
- на очень больших документах zoom может занимать заметное время, потому что страницы рендерятся заново;
- viewer остаётся лёгким и прагматичным, а не полноценным native PDF shell.

## Словари

- встроенный словарь ограничен и ориентирован на технические тексты;
- общий FreeDict зависит от доступности сети на этапе установки словаря;
- нет удаления словарей из GUI, только добавление / подключение.

## UI

- интерфейс построен на Tkinter, поэтому внешний вид улучшен стилями, но не дотягивает до полноценного modern native shell;
- настройка интерфейса сейчас ограничена размером шрифта.

## Платформы

- MVP протестирован как Linux desktop приложение;
- текущий Tkinter-based GUI не переносится на Android без отдельной UI-ветки.

## Перевод

- только словарный EN → RU;
- без контекстного нейронного перевода предложений;
- качество зависит от качества подключённых словарей и порядка их загрузки.
