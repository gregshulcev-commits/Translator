# app.py

## Назначение

Точка входа приложения.

## Ответственность

- разбор аргументов командной строки;
- подготовка runtime directories;
- сборка стартового словаря при необходимости;
- инициализация логирования;
- загрузка плагинов;
- создание сервисов и главного окна.
