# `document_service.py`

## Назначение

Сервис жизненного цикла документа и простого кэша отрендеренных страниц.

## Что делает

- открывает документ через активный `DocumentPlugin`;
- хранит текущий путь и `DocumentSession`;
- кэширует `RenderedPage` по `(page_index, zoom)`;
- предразогревает соседние страницы.

## Почему нужен отдельный сервис

UI не должен напрямую управлять сессией документа и кэшем.
