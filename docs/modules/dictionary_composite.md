# `dictionary_composite.py`

## Назначение

Объединяет несколько dictionary plugins в один логический источник lookup.

## Поведение

- ищет по plugins в заданном порядке;
- возвращает первый найденный перевод;
- если перевод не найден, возвращает fallback-result от первого словаря.

## Зачем нужен

Позволяет держать:

- маленький встроенный technical glossary;
- большой общий словарь;
- дополнительные пользовательские пакеты.

## Особенность v3

Добавлен `close()`, который пробрасывает закрытие дочерним plugins. Это нужно для аккуратного hot-reload словарного слоя.
