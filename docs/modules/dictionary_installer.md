# `dictionary_installer.py`

## Назначение

Набор helper-функций для установки словарных паков из GUI и CLI.

## Поддерживаемые сценарии

- подключение готового SQLite-пака;
- импорт CSV;
- импорт FreeDict TEI;
- установка пакета из catalog entry;
- установка набора паков для конкретного направления.

## Важная роль в v4

Через этот модуль GUI теперь умеет открывать **каталог пакетов** и устанавливать как bundled technical / literary packs, так и скачиваемые FreeDict packs.
