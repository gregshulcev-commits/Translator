# utils/text_normalizer.py

## Назначение

Подготовка английского слова к словарному lookup без тяжелых NLP-зависимостей.

## Что делает

- убирает внешнюю пунктуацию;
- приводит к lower-case;
- генерирует candidate forms;
- делает простые эвристики для plural / ed / ing / apostrophe forms.

## Почему не используется heavy NLP

Чтобы MVP оставался легким и полностью офлайн.
