# Roadmap

## Сделано в MVP v3

- просмотр PDF;
- просмотр TXT и FB2;
- клик по слову и выделение;
- нижняя компактная карточка перевода;
- поиск по документу;
- непрерывная прокрутка всего документа;
- zoom по `Ctrl + колесо мыши`;
- zoom до 600%;
- встроенный технический словарь;
- поддержка нескольких словарных паков;
- импорт словарей из GUI и через CLI;
- настройка размера интерфейса;
- автотесты и GUI smoke test.

## Ближайший следующий шаг (desktop)

- улучшить quality ranking при нескольких найденных словарях;
- сохранить last-opened document и last-view position;
- добавить историю слов как отдельный plugin/service;
- улучшить rendering performance больших документов (виртуализация страниц);
- добавить массовый hot-reload и удаление словарей из GUI.

## v1.1

- OCR-plugin;
- постраничное распознавание;
- редактирование OCR-результата пользователем;
- поддержка текста на изображениях;
- системный clipboard helper;
- простая карточка истории слов.

## v1.2

- EPUB / DOCX plugins;
- RU → EN;
- пользовательские профили словарей;
- экспорт незнакомых слов.

## v2

- отдельный translation plugin layer для фраз и предложений;
- нейронный перевод как optional pack;
- смена GUI backend-а (Qt/QML, Kivy или другой мобильный стек);
- Android-ветка на отдельном UI-слое.
