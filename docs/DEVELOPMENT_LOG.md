# Журнал разработки

## Итерация v2

### Изменения

- исправлен расчёт `PROJECT_ROOT`;
- убрана зависимость `pronouncing` и связанная с ней проблема `pkg_resources`;
- UI упрощён: нижняя компактная карточка перевода вместо перегруженной боковой панели;
- добавлена прокрутка колесом мыши;
- добавлен `CompositeDictionaryPlugin`;
- добавлен импорт FreeDict TEI и установщик словарей;
- обновлён CMake для корректной работы с `.venv`;
- расширен встроенный технический словарь;
- добавлены тесты на конфиг и FreeDict importer.

### Отдельно проверено

- `pytest` проходит;
- GUI smoke test проходит;
- реальные PDF пользователя успешно дают text tokens;
- fallback при отсутствии сети работает: приложение не остаётся без словаря.

### Честное ограничение этой итерации

В sandbox не было прямого исходящего сетевого доступа для самого установочного Python-скрипта, поэтому полный online-download общего словаря не удалось проверить end-to-end в той же среде, где собирался архив. В результате:

- логика downloader/importer реализована;
- парсер TEI покрыт unit-тестом;
- fallback-путь проверен;
- реальное скачивание словаря должно выполняться уже на машине пользователя во время установки.
