# `document_pdf_pymupdf.py`

## Назначение

Плагин документа для PDF на базе PyMuPDF.

## Возможности

- открытие PDF;
- рендер страницы в PIL Image;
- извлечение word tokens с координатами;
- поиск слова под курсором;
- сборка предложения вокруг токена;
- поиск по документу.

## Ограничение

Работает только для PDF с текстовым слоем. OCR не входит в текущую реализацию.
