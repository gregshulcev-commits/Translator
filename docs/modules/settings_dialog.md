# `settings_dialog.py`

## Назначение

Отдельное окно настроек desktop-приложения.

В `v9` это основной GUI-центр конфигурации, через который пользователь управляет провайдерами, интерфейсом, Argos и словарями без терминала.

## Структура окна

Окно построено как `ttk.Notebook` с вкладками:

- `Общие`
- `LibreTranslate`
- `Yandex Cloud`
- `Argos`
- `Словари`

## Что делает каждая вкладка

### Общие

- направление перевода `EN ↔ RU`;
- активный контекстный провайдер;
- размер UI-шрифта.

### LibreTranslate

- базовый URL или полный endpoint;
- API key;
- диагностика и проверка направлений `EN ↔ RU`.

### Yandex Cloud

- `Folder ID`;
- `API key`;
- `IAM token`.

### Argos

- установка optional Python runtime;
- обновление индекса моделей из сети;
- установка модели для выбранного направления;
- импорт локального `.argosmodel`;
- переключение активного провайдера на Argos.

### Словари

- просмотр установленных паков;
- открытие каталога словарей;
- подключение SQLite;
- импорт CSV;
- импорт FreeDict TEI;
- удаление пользовательских runtime-словарей.

## Почему модуль отделён от `MainWindow`

Вынесение в отдельный модуль уменьшает связность главного окна и позволяет развивать настройки как самостоятельный UX-узел без разрастания `main_window.py`.
