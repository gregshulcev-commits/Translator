# `dictionary_catalog.py`

## Назначение

Статический каталог известных словарных паков, которые приложение умеет устанавливать.

## Что описывает `DictionaryPackSpec`

- `pack_id`
- название;
- направление;
- категорию;
- install mode;
- локальный CSV-путь или сетевые URL.
