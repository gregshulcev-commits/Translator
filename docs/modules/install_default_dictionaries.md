# `install_default_dictionaries.py`

## Назначение

Подготовить usable dictionary setup после установки проекта.

## Что делает скрипт

1. собирает встроенный EN→RU словарь;
2. собирает встроенный RU→EN словарь;
3. устанавливает bundled technical / literary packs;
4. пытается скачать и импортировать FreeDict EN→RU;
5. пытается скачать и импортировать FreeDict RU→EN;
6. при отсутствии сети не падает, а завершает установку с предупреждением.
