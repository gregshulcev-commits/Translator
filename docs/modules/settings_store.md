# `settings_store.py`

## Назначение

Сохранение и загрузка пользовательских UI-настроек в JSON.

В `v9` этот модуль стал опорой для отдельного окна настроек: GUI больше не требует редактировать параметры вручную или держать их только в памяти текущего сеанса.

## Что хранится

- размер шрифта интерфейса;
- направление перевода;
- выбранный контекстный провайдер;
- URL / API key для LibreTranslate;
- credentials для Yandex Cloud.

## Что важно в `v9`

- значения по-прежнему нормализуются перед сохранением;
- `settings.json` сохраняется атомарно через временный файл и `os.replace()`;
- на POSIX файл получает права `0600`;
- новое окно `SettingsDialog` сохраняет изменения именно через этот слой.

Это не превращает JSON-файл в полноценное secret storage, но заметно уменьшает риск случайного раскрытия настроек провайдера другим пользователям системы.

## Почему это отдельный модуль

Разделение на `UiSettings` + `SettingsStore` упрощает:

- тестирование нормализации значений;
- переиспользование настроек в GUI;
- безопасное сохранение без дублирования файловой логики в `main_window.py`.
